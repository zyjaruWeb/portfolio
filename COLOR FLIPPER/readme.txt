
Project Name:

COLOR FLIPPER

Description:

Two webpages, one for a few simple colors, which are pre-stored in the program.
Second page, for randomly generated Hex colours.

Main Functions:

1. Array of pre-stored colours
2. function for when button clicked, random colour from simple colour array is choosen
3. the choosen colour then replaces the current in CSS style, and colour name sent to HTML to be displayed.
4. Function for hex colour to be randomly generated transformed into string 
and then sent to CSS style with an appropiate format
adding "#"
5. Randomly generated hex is displayed as text
